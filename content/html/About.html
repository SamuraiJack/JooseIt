VERSIONS

Joose is currently presented with 2 major releases: Joose 2 and Joose 3

Joose 2

Joose 2 is a stable, feature-rich and proven-to-work system. It is in production on several multi M$ sites (in both visitors and revenue), like:

http://www.tuifly.com/en/

http://www.mirapodo.de/

Joose 3

Joose 3 is currently under very active development. It is already stable with about 800 unit tests, though it may miss some features, presented in Joose 2 (like type system, serialization and some features for attributes).

On top of the Joose 2 feature set Joose 3 offers a more to-the-spec implementation of roles based on original traits spec, improved namespaces, better meta-roles support and mutability.

Joose 3 supports all modern browsers (including IE6) and the NodeJS platform.

Joose 3 is being distributed via the "npm" package manager and the OpenJSAN platform.
